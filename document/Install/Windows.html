# 安装浏览器

安装下方任意一个浏览器, 该浏览器在下文称为 浏览器:

  • Edge在新窗口打开

  • FireFox在新窗口打开

  • Chrome在新窗口打开

# 安装 Tampermonkey 油猴插件 (电脑端)

使用 浏览器 访问下方的商店页面, 并安装插件

  • Edge -> 微软商店在新窗口打开

  • FireFox -> 火狐商店在新窗口打开

  • Chrome -> 谷歌商店在新窗口打开 (可能无法访问)

# 安装脚本 (电脑端)
